1 # DMC32 OSU {#dmc32osu}
2 
3 \note gclib provides the communications foundation for the *DMC32 Operating System Upgrade (OSU)* project.
4 
5 DMC32 OSU is intended for existing applications that used software based on the legacy DMCWIN32 library for Windows XP and earlier. If such an application must be upgraded to Windows 7, 8, or 8.1, DMC32 OSU may be used on these O.S. upgrades.
